# ⚾ MLB Projections System

Sistema automatizado de análisis y predicción de estadísticas de la MLB.

## 🛠️ Tecnologías
- **Python 3.11**
- **LightGBM** para modelos predictivos.
- **Pandas** para procesamiento de datos.
- **GitHub Actions** para automatización total.

## 🚀 Cómo funciona
El sistema limpia automáticamente los datos, eliminando columnas no numéricas (como nombres de jugadores) antes de alimentar el modelo de Machine Learning, evitando errores de ejecución.
